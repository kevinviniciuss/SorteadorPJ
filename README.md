## 🎲 Sorteio
**Experimente agora 👉[Clicando Aqui]()**

## Sobre
Criação de um site onde o usuário pode adicionar participantes e realizar sorteios de forma rápida e visualmente atraente, com animações e um modo de apresentação ideal para uso de telas maiores.

## Funcionalidades
➕ Adicionar nomes ou números <br>
📋 Listagem dinâmica dos participantes <br>
❌ Remover itens da lista <br>
🎲 Sorteio aleatório <br>
🎬 Animação durante o sorteio (efeito suspense) <br>
🖥️ Modo apresentação (tela limpa para exibição) <br>

## 🎯Objetivo
Esse projeto foi desenvolvido com o objetivo de praticar:
<ul>
    <li>Manipulação do DOM</li>
    <li>Eventos em JavaScript</li>
    <li>Lógica de Programação</li>
    <li>Criação de Interfaces Interativas</li>
</ul>
